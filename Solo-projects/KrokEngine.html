KrokEngine

As a challenge and a learning experiance I tried making my own C++ 2D engine from scratch.

As it currently is it uses a Library called SFML, a simple and Fast Multimedia Library designed to provide a simple application programming interface to develop games and multimedia applications.

The purpose was to learn more about designing a game engine, memory menagment and ownership.

Next to this I wanted to try impliment 2D collision handeling for shapes considing of lines and circles.

The end goal is to be able to keep improving it and implimenting new features to be able to use it in, for instance, GameJams and keep learning more.
