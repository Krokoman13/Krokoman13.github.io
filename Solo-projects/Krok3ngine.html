Krok3ngine - Low level 3D DirectX engine

As a challenge and a learning experience I tried making my own C++ 2D engine from scratch.

The purpose was to learn more about DirectX, 3D engines, and the Windows API.


The engine has been made as a dynamic library and is very low level.

I did not use helper libraries to make using DirectX or the Win32 API specifically to learn more about low-level engeine development.


It has custom error logging, basic lighting, OBJ importing, and the window is fully customizable.
